# Student Performance Analysis

## 📊 Project Overview
This project analyzes student academic performance to identify key factors affecting grades and detect at-risk students.
The goal is to support data-driven decision-making in education systems.

## 📁 Dataset
Source: Kaggle – Students Performance in Exams
Contains student demographic and academic records
Features include:
Gender
Race/Ethnicity
Parental Education Level
Lunch Type
Test Preparation Course
Math, Reading, Writing Scores

## 🛠 Tools Used
- Python
- Pandas
 - NumPy
 - Matplotlib
 - Seaborn
 - Google Colab

## 🎯 Objectives
- Analyze factors affecting student performance
- Identify at-risk students
- Detect at-risk students (low performers)
- Provide actionable recommendations

## 📈 Key Insights
- Test preparation improves scores
- Parental education influences performance
- Strong correlation between reading and writing scores
- Female students perform better in reading and writing
- A noticeable percentage of students fall into the at-risk category

## 📊 Visualizations
- Box plots
- Bar charts
- Heatmap
- Scatter plot
- Histogram

## 💡 Recommendations
- Make test preparation mandatory
- Provide academic support for weak students
- Improve parent involvement
- Focus on improving mathematics performance

# Conclusion

This analysis highlights the key factors affecting student performance and provides actionable insights that can help improve academic outcomes using data-driven strategies.
