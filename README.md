# Student Performance Analysis

## 📊 Project Overview
This project analyzes student academic performance to identify key factors affecting grades and detect at-risk students.

## 📁 Dataset
Students Performance in Exams (Kaggle)

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 🎯 Objectives
- Analyze factors affecting student performance
- Identify at-risk students
- Provide actionable recommendations

## 📈 Key Insights
- Test preparation improves scores
- Parental education influences performance
- Strong correlation between reading and writing scores

## 📊 Visualizations
- Box plots
- Bar charts
- Heatmap
- Scatter plot
- Histogram

## 💡 Recommendations
- Make test preparation mandatory
- Provide academic support for weak students
- Improve parent involvement
